## Linked Screen Flow

Login/Register → Dashboard → Doctor List → Appointment Booking → Appointment Confirmation → Patient Profile/Records

## Prototype Interactions

- Login button opens Dashboard.
- Book Appointment opens Doctor List.
- Doctor card opens Appointment Booking.
- Confirm Appointment opens Confirmation.
- View Records opens Patient Profile.
- Recommended transitions: slide for forward movement and fade for confirmation/profile screens.
